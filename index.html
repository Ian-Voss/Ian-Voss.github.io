Esta es la página donde muestro algunos de los proyectos que realicé en mi trayecto como estudiante y programador.

Raycasting con OpenGL

El proyecto se basa en la visualización de una escena con el método de renderizado Raycasting utilizado en juegos como Wolfenstein. Escrito en C y con OpenGL, casi la comunicación más directa y básica que se puede tener con la placa de video. Implementando un método para utilizar solo dos triángulos por pared, el cual necesito utilizar un algoritmo de resolución de sistemas de ecuaciones lineales.

Renderizado 3D

Un proyecto de renderizado 3D más tradicional con proyección, suficientemente dinámico como para renderizar las esferas con método Ray Tracing que se pueden ver en el video, y con iluminación.

Clon de Minecraft

Con el mismo código de renderizado 3D anterior, usando ruido de perlin y FBM para la creación de la geometría, tenemos un clon básico del juego Minecraft básico.

Simulador de particulas

Partículas atraídas por gravedad entre sí, con las partículas siendo más pesadas mientras más rojas son. La simulación tiene la capacidad de soportar 1000 partículas o más utilizando multiprocessing para acelerar los cálculos.

Comportamiento de persecución

Se trata de una versión modificada del modelo de agentes de persecución propuesto propuesto en el siguiente paper: https://www.red3d.com/cwr/steer/.

Se trata de una versión modificada del modelo La modificación permite que los agentes tracen una curva mientras persiguen al objetivo.

Interprete de formulas en C

Una aplicación escrita en C capaz de evaluar fórmulas matemáticas sin volver a compilar, sin uso de ninguna librería aparte de las incluidas en el lenguaje. Código disponible pronto.
